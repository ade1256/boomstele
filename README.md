
# Boomstele

Bot automatic play game booms

- Claim daily reward

- Auto upgrade boost

- Auto join tournaments trading

## How to run ?

Clone repository

```bash
git clone https://github.com/ade1256/boomstele.git
cd boomstele
```

```bash
npm install
```

```bash
npm run start
```
